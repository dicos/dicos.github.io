Перенос postgis базы данных на другой сервер

  • By admin
  • Вс 02 Январь 2011

Начав востанавливать базу данных из бекапа увидел на экране кучу ошибок. Причиной ошибок оказался неподключенный к восстанавливаемой базе данных plpgsql язык программирования. Подключить его можно командой из bash консоли:

createlang plpgsql you_database
