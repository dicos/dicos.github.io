Установить локаль с поддержкой русских символов

export LC_ALL=ru_RU.UTF8
apt-get install language-pack-ru-base
dpkg-reconfigure locale

Для компиляции исходников: apt-get install libc6-dev g++ gcc

ngnix:

aptitude install libssl-dev libpcre3-dev

Postgres:

apt-get install libreadline-dev

запуск постгреса /usr/local/pgsql/bin/postgres -D /usr/local/pgsql/data

Редктировать, чтобы слушал от всех адресов и щифровал пароли
isten_addresses = '*'
password_encryption = on
Сборка питона:
aptitude install build-essential libsqlite3-dev libreadline5-dev libncurses5-dev zlib1g-dev libbz2-dev libssl-dev libgdbm-dev tk-dev

apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 314DF160 https://launchpad.net/~ubuntugis/+archive/ubuntugis-unstable
