Как определить кодировку строки в питоне

  • By admin
  • Сб 15 Май 2010

Для этого создана библиотека chardet. Мне она пригодилась для написания парсера писем, приходящих по рассылке в тогда неизвестной мне кодировке.

Для того, чтобы ею воспользоваться, нужно распаковать архив, и потом выполнить команду python setup.py с правами рута. Если не охота это делать, то можно просто скопировать каталог chardet в рабочую папку.
